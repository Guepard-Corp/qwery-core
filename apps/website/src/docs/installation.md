Install Qwery using one of the following options.

## Prerequisites

- Node.js >= 22.x
- pnpm >= 10.x

## From source

```bash
git clone https://github.com/Guepard-Corp/qwery-core.git
cd qwery-core
pnpm install
pnpm dev
```

The web app will be available at `http://localhost:3000`.

## Desktop application

```bash
pnpm desktop:dev
```
